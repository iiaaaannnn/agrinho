# Sustentabilidade Agrinho 2026: Transformando o Milho 🌽

Este projeto é uma página web informativa desenvolvida para o concurso Agrinho 2026. O objetivo é educar o público sobre o processo de transformação do milho em bioplástico, destacando inovação tecnológica e sustentabilidade no campo.

## 🚀 Tecnologias Utilizadas
- HTML5 (Semântico e Acessível)
- CSS3 (Variáveis e Flexbox)
- JavaScript (Manipulação de DOM)

## 🛠️ Configuração
Para visualizar o projeto localmente:
1. Clone este repositório ou baixe os arquivos.
2. Certifique-se de que a estrutura de pastas contém: `index.html`, `style.css` e `script.js`.
3. Abra o arquivo `index.html` em qualquer navegador moderno.

## 📖 Como usar
- Navegue pelas seções de **Benefícios** e **Detalhes Técnicos**.
- Utilize o botão "Saiba Mais" para disparar informações adicionais via console ou alerta.
- A página é totalmente responsiva e acessível via teclado.
